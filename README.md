usage: pubupd [-h] [-c] [-a] [-u] dir

Utility to update dependencies in flutter project's pubspec.yaml

positional arguments:
  dir                   The directory of flutter project containing
                        pubspec.yaml

optional arguments:
  -h, --help            show this help message and exit
  -c, --change-log      Display changelog
  -a, --display-all     Display all packages including updated ones
  -u, --update-pubspec  Update pubspec.yaml file with latest versions
